pratyaksha.sh — terminal portfolio.

built with next.js, three.js, react-three-fiber, rapier physics, and a questionable amount of green.

```bash
npm install
npm run dev
```

-  3D lanyard id card with physics (drag it)
-  linux-style terminal with 12 commands
-  60 particles drifting upward (because why not)
-  themes: hacker, matrix, neon, retro, light
-  zero social life, full send
